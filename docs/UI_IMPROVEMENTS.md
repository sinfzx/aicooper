## 📊 技术实现

### 组件架构
```
PageLayout (通用布局)
├── 面包屑导航
├── 页面头部 (标题、描述、图标)
├── 操作按钮组
└── 页面内容区域

功能页面
├── Manager 组件 (列表页)
├── Detail 组件 (详情页)
└── Modal 组件 (创建/编辑)
```

### 路由结构
```
/                    # 主页面 (单页应用)
├── /knowledge/[id]  # 知识框架详情
├── /templates/[id]  # 提示词模板详情
└── /flows/[id]      # 流程设计详情
```

### 样式系统
- **组件库**: Mantine UI
- **图标**: Tabler Icons
- **主题**: 统一的颜色和间距系统
- **响应式**: 支持不同屏幕尺寸
