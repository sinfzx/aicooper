## 🚀 如何运行项目

### 1. 环境准备
```bash
# 确保已安装 Node.js 18+ 和 Rust
node --version
cargo --version
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
cd apps/desktop
npm run tauri:dev
```

### 4. 构建生产版本
```bash
cd apps/desktop
npm run tauri:build
```
